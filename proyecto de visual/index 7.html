Explorando el Mundo de HTML

La base fundamental de toda página web

¿Internet y la Web son lo mismo?

Aunque muchas personas usan estos términos indistintamente, Internet y la Web no son lo mismo.

Internet

Es la infraestructura global de redes interconectadas que permite la comunicación entre dispositivos. Incluye:

  • Correo electrónico (SMTP)
  • Transferencia de archivos (FTP)
  • Mensajería instantánea
  • Voz sobre IP (VoIP)

World Wide Web

Es un servicio que funciona sobre Internet, que permite acceder a documentos hipertextuales (páginas web) a través de navegadores.

  • Basado en HTTP/HTTPS
  • Utiliza navegadores web
  • Documentos enlazados con hipervínculos
  • Creado por Tim Berners-Lee en 1989
// Analogía común:
Internet = Sistema de carreteras
Web = Autobuses y camiones que transportan información
Navegadores = Vehículos que usamos para acceder a la web

Componentes Web Esenciales

URL

Uniform Resource Locator - La dirección única que identifica un recurso en la web.

https://www.ejemplo.com/ruta/pagina.html#seccion?parametro=valor

Navegador Web

Software que interpreta y muestra páginas web (Chrome, Firefox, Safari, Edge).

Buscador

Sistema que indexa y permite buscar información en la web (Google, Bing, DuckDuckGo).

Servidores

Computadoras que almacenan y sirven páginas web a los navegadores cuando son solicitadas.

Codificación HTML

HTML se basa en el concepto de hipertexto - texto que contiene enlaces a otros textos.

¿Cómo codificamos HTML?

Editores de texto

Software especializado para escribir código:

  • Visual Studio Code
  • Sublime Text
  • Atom
  • Notepad++

Editores WYSIWYG

Editores visuales que generan el código:

  • Adobe Dreamweaver
  • WordPress (para contenido)
  • Webflow

Estructura básica

Todos los documentos HTML tienen:

  1. Declaración DOCTYPE
  2. Elemento <html>
  3. Sección <head>
  4. Sección <body>

Tipos de Etiquetas HTML

Etiquetas de Encabezado

<h1> a <h6> - Definen títulos y subtítulos.

<h1>Título principal</h1>
<h2>Subtítulo</h2>

Etiquetas de Texto

<p>, <span>, <strong>, <em> - Formatean texto.

<p>Este es un <strong>párrafo</strong>.</p>

Etiquetas de Sección

<div>, <section>, <article>, <header>, <footer> - Estructuran el contenido.

Etiquetas de Enlace

<a> - Crea hipervínculos.

<a href="pagina.html">Enlace</a>

Etiquetas Multimedia

<img>, <video>, <audio> - Incrustan medios.

Etiquetas de Tabla

<table>, <tr>, <td>, <th> - Crean tablas.

Iframes

<iframe> - Incrusta otra página web.

<iframe src="otra_pagina.html"></iframe>

Atributos y Valores en HTML

Atributos comunes

Atributo Descripción Ejemplo
id Identificador único para un elemento <div id="header">
class Clase CSS para aplicar estilos <p class="destacado">
style Estilos CSS en línea <h1 style="color:red;">
src Fuente para imágenes, scripts, etc. <img src="logo.png">
href URL para enlaces <a href="contacto.html">
alt Texto alternativo para imágenes <img alt="Logo empresa">

Formas de aplicar estilos

1. Estilos en línea

Usando el atributo style directamente en el elemento.

<p style="color: blue;">Texto azul</p>

2. Hoja de estilos interna

Usando la etiqueta <style> en el <head>.

<style>
  p { color: blue; }
</style>

3. Hoja de estilos externa

Enlazando un archivo CSS externo.

<link rel="stylesheet" href="estilos.css">

Propiedades de Texto y Tipografías

Propiedades de texto

  • color: Color del texto
  • font-family: Tipo de letra
  • font-size: Tamaño del texto
  • font-weight: Grosor (normal, bold)
  • text-align: Alineación (left, center, right)
  • line-height: Espacio entre líneas
  • text-decoration: Subrayado, etc.

Tipografías web

Fuentes pueden ser:

  • Seguras para web: Arial, Times, Verdana
  • Google Fonts: Fuentes personalizadas
  • @font-face: Fuentes personalizadas
<link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">

Templates o Plantillas en HTML

Las plantillas HTML son estructuras predefinidas que permiten reutilizar código y mantener consistencia en un sitio web.

Plantillas básicas

Estructuras HTML simples que sirven como punto de partida para páginas nuevas.

Frameworks

Bootstrap, Foundation y otros ofrecen plantillas predefinidas.

Componentes reutilizables

Headers, footers, cards y otros elementos que pueden copiarse entre páginas.

Ventajas de usar plantillas

  • Consistencia en el diseño
  • Ahorro de tiempo en desarrollo
  • Más fácil mantenimiento
  • Responsive design probado

Alojamiento y Publicación de Sitios Web

Tipos de hosting

  • Compartido: Varios sitios en un mismo servidor
  • VPS: Servidor virtual privado
  • Dedicado: Servidor físico completo
  • Cloud: Alojamiento en la nube

Publicación

Pasos para publicar un sitio:

  1. Elegir proveedor de hosting
  2. Registrar dominio
  3. Subir archivos vía FTP o administrador
  4. Configurar DNS

Herramientas

Software útil para publicación:

  • FileZilla (FTP)
  • Git/GitHub Pages
  • cPanel
  • Firebase Hosting
